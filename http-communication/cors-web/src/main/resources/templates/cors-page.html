CORS 테스트 페이지

1. 단순요청 - 버튼을 클릭하여 http://localhost:10000/cors/simple-request에서 데이터를 가져옵니다.

단순요청 결과가 여기에 표시됩니다.

2. 사전 검증 요청 - 버튼을 클릭하여 http://localhost:10000/cors/preflighted-request에 커스텀 헤더와 함께 요청을 보냅니다.

사전 검증 요청 결과가 여기에 표시됩니다.

3. 자격증명을 포함한 요청 - 버튼을 클릭하여 http://localhost:10000/cors/credentialed-request에 쿠키와 자격증명을 포함하여 요청을 보냅니다.

자격증명 요청 결과가 여기에 표시됩니다.
